图片放大镜效果, 功能说明:

  • 镜片模式: 鼠标移到那里的一个特定范围内的图片被放大显示, 显示框在原图旁边;
  • 跟随模式: 显示框跟随鼠标移动, 即被放大的图片显示在原图上方, 并以鼠标为轴心;

ImageZoom Method

Object ImageZoom(img, cfg)()

根据给定的选择器img, 按照配置cfg, 返回ImageZoom对象

ImageZoom Config

config - Object


defaultConfig = {
    type: 'default',    // 显示模式, 可选值: TYPE

    bigImageSrc: '',    // 大图路径, 为 '' 时取原图路径
    bigImageSize: { height: 900, width:900 }, // 大图高宽
    position: 'right',  // 大图显示位置, 可选值: POSITION
    offset: 10,         // 大图位置的偏移量
    preload: true,      // 是否预加载大图
    timeout: 6000,      // 等待大图加载的最大时间, 单位: ms, 
                        // 当超过timeout时间之后, 图片仍未加载, 
                        // 会给出提示, 如果之后图片加载成功会覆盖提示;

    glassSize: { height: 100, width: 100 }, // 镜片高宽
    zoomIcon: true      // 是否显示放大镜提示图标
};

使用示例: 


<body>
<div id="page">
    <img id="smallimg" class="jqzoom" src="close1_small.jpg" />
</div>
</body>

<script>
KISSY.ready(function(S) {
    var zm = S.ImageZoom("#smallimg", // 提供被放大图片选择器
        {bigImageSrc:"close1_big.jpg"});
});
</script>


<!-- 构建出的DOM -->
<body>
<div id="page">
    <div class="ks-imagezoom-magnifier" data-ks-event-guid="1277113364450">
        <img src="close1_small.jpg" class="jqzoom" id="smallimg">
        <div class="ks-imagezoom-glass hidden"></div>
        <div class="ks-imagezoom-icon"></div>
    </div>
</div>
<div class="ks-imagezoom-viewer hidden">
    <img src="close1_big.jpg">
</div>
</body>

特别注意：1. 跟随模式下生成的dom有点不同, 直接把ks-imagezoom-viewer这层放到ks-imagezoom-magnifier中; 2. 关于显示区域宽高, 显示区域宽=大图宽度*镜片宽度/小图宽度;
