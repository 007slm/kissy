ua.js

通过 KISSY.UA 的属性，你可以获取浏览器等用户代理的信息。属性值遵循以下规则：

  1. 表示当前引擎或浏览器的版本。版本号 1.2.3.4 会转换为数值 1.234
  2. 如果不是当前引擎或浏览器，返回 0 或者 undefined
  3. 如果当前浏览器版本号无法准确判定，均返回 0.1

特别注意：

  1. KISSY.UA.mobile 返回的是字符串，目前支持 Apple, Nokia, Android 等设备的探测。

Properties

webkit - Number

webkit 的版本号。

chrome - Number

chrome 的版本号。

safari - Number

safari 的版本号。

gecko - Number

gecko 的版本号。

firefox - Number

firefox 的版本号。

ie - Number

ie 的版本号。

opera - Number

opera 的版本号。

mobile - String

mobile 的标志符。

se360 - Number

360 安全浏览器的版本号。

目前360安全浏览器2.x系列的版本无法识别，只有3.x系列的才能正常识别，因此目前只要识别为360安全浏览器，都返回3.0版本号。此返回值可能不正确。

maxthon - Number

maxthon 的版本号。

目前最新的傲游浏览器（3.x系列）采用双核技术，因此在版本号识别上可能导致错误，因此若返回值为0.1则表示当前傲游浏览器的版本号无法确定，其余版本则不存此问题。

tt - Number

腾讯TT浏览器的版本号。

theworld - Number

世界之窗浏览器的版本号。

目前世界之窗浏览器2.x系列的版本无法识别，只有3.x系列的才能正常识别，因此目前只要识别为世界之窗浏览器，都返回3.0版本号。此返回值可能不正确。

sougou - Number

搜狗浏览器的版本号。

rawie - Number or undefined

原生IE的版本号。

如果用户使用的原生IE浏览器，则返回此IE浏览器的版本号，若没有使用原生IE浏览器，则返回undefined。

返回值可能无法完全保证当前浏览器一定为原生IE。

shell - String or undefined

如果用户浏览器使用的是国产第三方浏览器，无论是IE内核或者Webkit内核，均可识别，并返回浏览器的字符串形式，与上面的第三方国产浏览器接口保持了一致。在无法确定是否为国产第三方浏览器时，返回undefined
