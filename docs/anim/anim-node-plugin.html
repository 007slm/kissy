anim-node-plugin.js

anim-node-plugin 通过将 Anim 耦合到 Node/NodeList 上，让用户能更简单快捷的调用。

直接看范例：


KISSY.ready(function(S) {
    S.one('#test-btn').on('click', function() {
	    S.all('.test').animate('left: 800px', 5, 'elasticOut');
    });
});

点击 #test-btn 时，就为所有 .test 元素添加了动画效果。

没错，一切就这么简单！
