Flash模块对 KISSY.UA 进行了扩展，规则与 KISSY.UA 类似，其中包括两个属性：

  1. KISSY.UA.fpv 返回的是Flash版本号的字符串，例如 10.1.53。
  2. KISSY.UA.fpvGEQ 用来检测当前客户端版本是否高于目标版本

注意，fpv是Flash Player Version的简写。

Properties

fpv - Number

Flash 的版本号。

fpvGEQ - Boolean

当前Flash版本号是否高于目标版本。

Flash Embed组件提供了对Flash添加写入的支持。

Examples

动态添加Flash到指定Dom：


var S = KISSY, F = S.Flash;

F.add('#Foo', { //目标Dom节点，将会被替换为Flash对象
    src: 'assets/test.swf', // swf 路径
    version: 9, //要求的 Flash Player 最低版本
    attrs: { // swf 对应 DOM 元素的属性
        width: 200, // 最小控制面板宽度,小于此数字将无法支持在线快速安装
        height: 150 // 最小控制面板高度,小于此数字将无法支持在线快速安装
    },
    params: {
        flashvars: { // Flash Player 的配置参数
            s: "string",
            b: false,
            n: 1,
            nul: null,
            und: undefined,
            url: "http://taobao.com/?x=1&z=2",
            o: {
                s: "string",
                b: false,
                n: 1,
                url: "http://taobao.com/?x=1&z=2"
            }
        }
    }
});

快速安装Flash player：


var S = KISSY, F = S.Flash;

F.add('#Foo', { //目标Dom节点，将会被替换为Flash对象
    src: 'assets/test.swf', // swf 路径
    xi: 'express-install/expressInstall.swf', //快速安装地址。全称 express install
    version: 12.2, //	要求的 Flash Player 最低版本
    params: {
        flashvars: 'a=1&b=2'
    }
});
