flash-ua.js

flash-ua 模块给 S.UA 增加了获取客户端 Flash Player 版本信息的方法。

Methods

fpv Array fpv( force )

返回当前客户端 Flash Player 的版本号数组，格式为 [ Major, Minor, Revision ]

force 参数表示是否强制重新获取版本信息。

fpv 是 Flash Player Version 的简写。

fpvGEQ Boolean fpv ( ver, force )

判断当前 Flash Player 版本号是否高于指定版本。


if(KISSY.UA.fpvGEQ('9.1.0') {
    // do sth.
}
